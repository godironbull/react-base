Bem-vindo ao seu aplicativo Expo 👋

Este é um projeto Expo criado com o create-expo-app


## Comece agora

1. Instale as dependências

   ```bash
   npm install
   ```

2. Inicie o aplicativo

   ```bash
   npx expo start
   ```

## Obtenha um projeto limpo

Quando estiver pronto, execute:

```bash
npm run reset-project
```

Este comando moverá o código inicial para o diretório app-example e criará um diretório app em branco onde você poderá começar a desenvolver do zero.


## 📁 Estrutura de Pastas

Uma breve explicação de como os arquivos do projeto estão organizados:

```text
├── src/
│   ├── assets/          # Imagens, ícones, logotipos e fontes
│   ├── components/      # Componentes visuais reutilizáveis
│   ├── hooks/           # Custom Hooks personalizados do React
│   ├── screens/         # Telas ou páginas principais da aplicação
│   ├── services/        # Configurações de serviços externos e APIs
│   ├── styles/          # Estilizações globais ou configurações de temas
│   └── App.tsx          # Componente raiz da aplicação
├── .gitignore           # Ficheiros e pastas ignorados pelo Git (ex: node_modules)
├── package.json         # Dependências, metadados e scripts do projeto
└── README.md            # Este ficheiro de documentação
```
